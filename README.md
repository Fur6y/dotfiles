# Dotfiles

## Install

- run 'makesymlinks.sh'

## Aliases

```
mkd             # create a new directory and enter it
ls              # list files
la              # list files and hidden files
ll              # list files and hidden files as list
update          # runs update for osx, brew, brew cask, npm, gem
ipglobal        # print global ip address
ip              # print local ip address
nl              # print list of all local top level npm dependencies
ng              # print list of all global top level npm dependencies
ndev            # set node environment variable to development
nprod           # set node environment variable to production
nstage          # set node environment variable to staging
defaultshell    # set bash as default shell
fishshell       # set fish as default shell
aliases         # print list of all aliases
```
